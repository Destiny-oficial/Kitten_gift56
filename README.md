pkg install git

pkg update

pkg upgrade

pkg install ffmpeg

pkg install wget

pkg install nodejs

pkg install npm

npm install

instala la dependencia 
npm install @whiskeysockets/baileys @hapi/boom node-cache readline awesome-phonenumber cfonts pino fs axios qrcode --no-bin-links

inicia el bot
node index2.js
